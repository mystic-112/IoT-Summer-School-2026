# Branching Strategy

## main
Contains the stable and tested version of the project.

## dev
Contains ongoing development, documentation, and integration work before merging into main.

## sensor-module
Contains all sensor-related development such as DHT11 interfacing, data acquisition, and sensor processing.

## wireless-module
Contains Bluetooth Low Energy (BLE) communication, GATT services, characteristics, descriptors, and wireless data transmission.
